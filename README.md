# Portfólio 

Bem vindo ao meu portfólio.

## Layout web:

A página conta com 3 abas diferentes, sendo a primeira uma rápida apresentação com alguns meios de contatos, a segunda uma apresentação mais aprofundada sobre mim e a terceira o meu currículo.

![image](https://raw.githubusercontent.com/joaovictormar/portfolio/main/assets/web.jpeg)

## Layout mobile:

No layout mobile foi mantido tudo do layout web, alterando apenas algumas posições de elementos e tamanhos, fazendo com que a página fique responsiva.

![image](https://raw.githubusercontent.com/joaovictormar/portfolio/main/assets/mobile.jpeg)

## Objetivo do projeto:

O objetivo aqui foi desenvolver um pequeno portfólio, para que todos me conheçam um pouco melhor. 
Nesse projeto irei manter atualizado tudo que eu adquirir, a cada novo certificado, experiência e vivência que terei na minha vida profissional estará aqui.
